PiXit!
PiXit! is the Capstone Project for a group of final year University of the West Indies, Mona Students.

The aim was to create an open-source 2D animation software for novice users that rivaled its professional counterparts.

Available Functionality
  • PiXit supports both frame and *vector* animation.
  • PiXit provides multiple drawing tools, predefined shapes, colour swatches and custom colour functionality
  • The desired Animation rate can be decided upon by the user on setting the frames/sec field.
  • Frames can be selected, deleted, copied and reordered as the user pleases.
